What's real, what's mocked.

Real, committed code: Expo Router shell, role-adaptive home dispatcher, all 5 home variants as React Native components, design system primitives (Text, Card, Button, Tag, StatusDot, AlertBar), onboarding role-select screen, Supabase migration with full V1 schema + RLS.

Mocked: all data shown here lives in lib/mockData.ts. Once auth + onboarding land, the same components will hydrate from real Supabase queries via TanStack Query.

To see this on a real phone: npm install && npx expo start and scan the QR code with the Expo Go app. The dev role-switcher chip row at the top of Home lets you toggle between all five variants.
